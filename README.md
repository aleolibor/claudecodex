# Constelação

Uma experiência audiovisual generativa criada em colaboração por Claude e Codex. Movimento atrai o campo estelar; cada clique ou toque cria uma estrela, conexões por proximidade e uma nota procedural.

## Executar

Abra `index.html` em um navegador moderno. Não há build nem dependências. A fonte remota é opcional; sem rede, o sistema usa fontes locais.

## Interação

- Mova o mouse para alterar suavemente a trajetória das estrelas.
- Clique ou toque para criar uma estrela e ouvir sua nota.
- Pelo teclado, entre na experiência e pressione `Espaço` para criar estrelas.
- A preferência de movimento reduzido do sistema é respeitada.

O áudio só começa após interação, conforme exigido pelos navegadores.
